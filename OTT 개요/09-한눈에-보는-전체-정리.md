# 09 - 한눈에 보는 전체 정리

> 이 문서는 8개의 상세 문서를 **5분 만에 이해할 수 있도록** 압축한 요약이다.

---

## 📌 프로젝트 정체성

**"SAP BTP 위에서 OTT 스트리밍 플랫폼을 구축하는 엔터프라이즈 SaaS"**

- **이름:** Zen OTT Onboarding
- **유형:** SAP BTP Side-by-Side Extension 모노레포
- **백엔드:** SAP CAP + TypeScript + HANA + XSUAA
- **프론트엔드:** SAPUI5 (portal, sysmgt)
- **배포:** SAP BTP Cloud Foundry (jp10 리전)

## 🧩 핵심 비즈니스 로직

### 1. 권한 기반 메뉴 표시
```
User → User_RoleGroup → RoleGroup → Role_RoleGroup → Role → Role_Menu → Menu
```
사용자에게 허용된 메뉴만 포털에 표시. `MenuFunctionAppliedList` View로 UI 기능 제어.

### 2. 동적 i18n (DB 기반)
```
UI5 CoreResourceModel → GET /srv-api/i18n.properties
  → server.ts generateI18n()
    → SELECT FROM Message WHERE language_code = 'KO'
      → .properties 형식 변환 → 응답
```

### 3. 동적 메뉴 라우팅 (Portal)
```
MenuLoader → MenusRoleAppliedList → Component.js
  → menu_app_id별 Route/Target/ComponentUsage 동적 생성
  → lazy loading으로 실제 접근 시에만 로드
```

### 4. OTT 추세 분석 + 이상 탐지
```
calculateTrends(analysisMonth):
  1. SubscriberGroup 조회
  2. 그룹 타입별 사용자 필터링
  3. ViewingHistory 집계 (시청 시간)
  4. GroupTrend + ContentGroupStat INSERT
  5. 전월 비교 → 임계값 초과 시 TrendAnomaly 자동 생성
```

### 5. 정산 배분
```
totalRevenue = Σ(active 사용자 × 플랜 가격)
partnerShare = totalRevenue × (partnerWatchSeconds / totalWatchSeconds)
sharedRevenue = partnerShare × rsRate
```

### 6. 게시판 CRUD + 작성자 인증
```
CREATE: before → req.user.id 자동 주입
UPDATE: before → 작성자 확인 (403 if mismatch)
DELETE: before+on → 소프트 삭제 (deleted_flag=true)
READ: after → 단건 조회 시 view_count += 1
```

## ⚡ 개발 명령어 치트시트

```bash
# 설치
npm run init:all                    # 모든 의존성 설치

# 실행 (전체)
npm run watch                       # CAP core + AppRouter + UI5 모두

# 실행 (개별)
npm run cap                         # CAP core (8080)
npm run router                      # AppRouter (5000)
npm run ui5                         # UI5 dev server (5001)

# 빌드
cd cap-node/core && npm run build:cf
cd cap-node/ott && npm run build:cf

# 배포
npm run deploy:core                 # CAP core 배포
npm run deploy:router               # AppRouter 배포 (dev)
npm run deploy:sysmgt               # 시스템 관리 UI 배포
```
