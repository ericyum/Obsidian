# 5. Integration Suite 가격 정책

> **작성일**: 2026-07-20
> **출처**: SAP Discovery Center, SAP Store, 공식 문서 종합

---

## 과금 모델 개요

SAP Integration Suite는 **구독(Subscription)** 기반으로 과금되며, 크게 두 가지 과금 차원이 있습니다:

```
과금 체계
├── ① 라이선스 모델
│     ├── Standard Edition    (기본 기능)
│     ├── Enterprise Edition  (전체 기능 + B2B/EDI)
│     └── 개별 Capability 추가
│
├── ② 측정 기준 (Metering)
│     ├── 테넌트당 정액 (Tenant-based)
│     ├── 메시지 건수당 (Message-based / Pay-per-use)
│     └── 대역폭 기반 (Bandwidth-based)
│
└── ③ 추가 옵션
      ├── Non-Production 테넌트
      ├── 추가 어댑터
      └── 프리미엄 지원
```

## 구체적 가격 체계 (2026년 기준, 대략적)

> ⚠️ 실제 가격은 계약 조건, 지역, 볼륨 할인에 따라 달라집니다. 정확한 견적은 SAP 영업 담당자에게 문의 필요.

### Standard Edition

| 측정 단위 | 월간 기준 가격대 | 설명 |
|-----------|:---:|------|
| **테넌트당** | $7,000 ~ $12,000/월 | 기본 테넌트 1개 기준 |
| **추가 테넌트** (Non-Prod) | $2,000 ~ $4,000/월 | Dev/Test 환경용 |
| **메시지 1만 건당** | $150 ~ $300/월 | Pay-per-use 옵션 |

**Standard로 충분한 경우**:
- SAP ↔ SAP/Non-SAP 프로세스 통합
- API 관리·게시 필요
- B2B/EDI 없음
- 이벤트 메시징은 별도 구매 가능

### Enterprise Edition

| 측정 단위 | 월간 기준 가격대 | 설명 |
|-----------|:---:|------|
| **테넌트당** | $12,000 ~ $20,000/월 | 기본 테넌트 1개 |
| **추가 테넌트** (Non-Prod) | $4,000 ~ $6,000/월 | Dev/Test 환경용 |
| **대역폭 10GB당** | $1,000 ~ $2,000/월 | 데이터 전송량 기준 |

**Enterprise가 필요한 경우**:
- B2B/EDI 거래 파트너와 전자문서 교환
- 하이브리드 시나리오 (Edge Integration Cell)
- Event Mesh 포함 통합
- Integration Advisor 고급 AI 매핑

## 추가 비용 요소

Integration Suite 자체 라이선스 외에 **SAP BTP 기반 인프라 비용**이 추가될 수 있습니다:

| 요소 | 설명 | 비용 영향 |
|------|------|:---:|
| **BTP 계정** | Global Account / Subaccount | 일반적으로 무료 |
| **CF 런타임** | Cloud Foundry 메모리/인스턴스 | 개별 통합 실행 시 추가 |
| **Kyma 런타임** | Kubernetes 기반 이벤트 처리 | Event Mesh 시 추가 |
| **HANA Cloud** | 데이터 지속성 스토리지 | 대량 데이터 저장 시 |
| **Cloud Connector** | On-prem 연결 터널 | 무료 |
| **XSUAA** | 인증 서비스 | BTP 기본 포함 |

## 볼륨 할인 & 계약 모델

### CPEA (Cloud Platform Enterprise Agreement)

- 연간 최소 커밋먼트 기반 계약
- 사용량에 따라 유연하게 크레딧 소비
- 대기업·파트너사에 유리
- **할인율**: 볼륨에 따라 20~50%

### Subscription (구독)

- 1년/3년 단위 구독 계약
- 고정 월 비용, 예산 예측 용이
- **할인율**: 연간 계약 시 10~20%, 3년 계약 시 20~40%

### Pay-as-you-go

- 약정 없는 월 단위 과금
- 스타트업·PoC에 적합
- **할인 없음**, 가장 비싼 옵션

## 비용 예시 시나리오

### 시나리오 A: 중소규모 SAP 통합 (Standard)

```
Standard Edition × 1 Prod Tenants + 1 Non-Prod
  = ($10,000 + $3,000) × 12개월
  = $156,000/년 (약 2.1억 원)

메시지 건수 기반 추가 비용: 월 10만 건 → $150 × 10 = $1,500/월 추가
  → 연간 $18,000 추가

총 예상: $174,000/년 (약 2.3억 원)
```

### 시나리오 B: B2B EDI 포함 (Enterprise)

```
Enterprise Edition × 1 Prod Tenants + 1 Non-Prod
  = ($15,000 + $5,000) × 12개월
  = $240,000/년 (약 3.2억 원)

대역폭 100GB/월 → $1,500 × 10 = $15,000/월 추가
  → 연간 $180,000 추가

총 예상: $420,000/년 (약 5.6억 원)
```

## 비용 최적화 팁

| 전략 | 설명 | 예상 절감 |
|------|------|:---:|
| **3년 약정** | 장기 계약 할인 | 20~40% |
| **CPEA 활용** | 크레딧 유연 소비 | 20~30% |
| **Pre-built 활용** | 커스텀 개발 대신 SAP 제공 콘텐츠 사용 | 개발 비용 80%+ 절감 |
| **Dev Tenant 최소화** | 꼭 필요한 Non-Prod만 | $3~5,000/월 절감 |
| **메시지 배치** | 개별 전송 대신 배치 처리 | 메시지 비용 50%+ 절감 |
