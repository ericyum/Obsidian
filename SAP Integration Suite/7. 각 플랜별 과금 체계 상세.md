# 7. 각 플랜별 과금 체계 상세

> **작성일**: 2026-07-20

---

## 플랜 개요

SAP Integration Suite의 각 플랜은 **구독 라이선스 + 사용량 측정**의 2중 과금 체계를 가집니다:

```
과금 = 구독 라이선스 (고정) + 사용량 측정 (변동)
       ├── 테넌트 비용            ├── 메시지 건수
       └── Capability 포함 범위    ├── 대역폭
                                  ├── API 호출 수
                                  ├── 커넥터 수
                                  └── 데이터 저장량
```

---

## Standard 플랜

### 구독 라이선스 (고정)

| 항목 | 가격대 (월) | 설명 |
|------|:---:|------|
| **Production Tenant 1개** | $7,000 ~ $12,000 | 기본 구독, 1개 포함 |
| **추가 Production Tenant** | $7,000 ~ $12,000 | 필요 시 |
| **Non-Production Tenant** | $2,000 ~ $4,000 | Dev/Test/QA |
| **Sandbox Tenant** | $1,500 ~ $2,500 | 학습·PoC용 |

### 사용량 측정 (변동)

| 측정 기준 | 단위 | 가격대 | 설명 |
|-----------|------|:---:|------|
| **Message Processing** | 10,000건 | $150 ~ $300 | iFlow 메시지 처리 건수 |
| **API Calls** | 100만 호출 | $500 ~ $1,200 | API Management 호출 |
| **Data Transfer** (Outbound) | 10GB | $1,000 ~ $2,000 | 아웃바운드 데이터 전송 |
| **Connector Calls** | 1,000건 | $50 ~ $150 | Open Connectors 호출 |
| **Persistent Storage** | 1GB/월 | $5 ~ $10 | 메시지 데이터 저장 |

### Standard에 포함된 기본 용량

| 항목 | 기본 제공량 | 초과 시 |
|------|:---:|------|
| **Message Processing** | 월 50만 건 | 건당 추가 과금 |
| **API Calls** | 월 500만 호출 | 호출당 추가 과금 |
| **Data Transfer** | 월 50GB | GB당 추가 과금 |
| **Tenant Storage** | 50GB | GB당 추가 과금 |
| **Concurrent iFlow 실행** | 10~20개 (리전별) | 추가 불가 (기술 제한) |

## Enterprise 플랜

### 구독 라이선스 (고정)

| 항목 | 가격대 (월) | 설명 |
|------|:---:|------|
| **Production Tenant 1개** | $12,000 ~ $20,000 | 기본 구독, 1개 포함 |
| **추가 Production Tenant** | $12,000 ~ $20,000 | 필요 시 |
| **Non-Production Tenant** | $4,000 ~ $6,000 | Dev/Test/QA |
| **Sandbox Tenant** | $2,500 ~ $4,000 | 학습·PoC용 |

### Enterprise에만 포함되는 추가 기능

| 기능 | Standard | Enterprise |
|------|:--:|:--:|
| **Trading Partner Management** | ❌ | ✅ (Partner 50개 기본) |
| **추가 B2B Partner** | — | $100 ~ $300/Partner/월 |
| **Event Mesh Queue** | 별도 | ✅ (Queue 5개 기본) |
| **Event Mesh Topic** | 별도 | ✅ (Topic 10개 기본) |
| **Integration Advisor MIG** | 제한 | ✅ (MIG 10개 기본) |
| **Integration Advisor MAP** | ❌ | ✅ (AI 매핑 제안) |
| **Edge Integration Cell Node** | ❌ | ✅ (Node 2개 기본) |

### Enterprise 사용량 측정 (Standard 대비 높은 기준)

| 측정 기준 | 기본 제공량 | 비고 |
|-----------|:---:|------|
| **Message Processing** | 월 100만 건 | Standard의 2배 |
| **B2B Messages** | 월 10만 건 | EDI 메시지 별도 |
| **API Calls** | 월 1,000만 호출 | Standard의 2배 |
| **Event Mesh Messages** | 월 500만 건 | 이벤트 메시지 별도 |

## 과금 예시 실제 분석

### 케이스 1: 중소기업 SAP SuccessFactors 연동 (Standard)

```
월별 비용 내역:
  Production Tenant:     $10,000
  Dev Tenant:            $3,000
  Message Processing:    30만 건 (기본 50만 건 포함) → $0 추가
  API Calls:             200만 호출 (기본 500만 호출 포함) → $0 추가
  Data Transfer:         20GB (기본 50GB 포함) → $0 추가
────────────────────────────────
  월 총계: $13,000
  연간: $156,000 (약 2.1억 원)
```

### 케이스 2: 대기업 B2B EDI 구축 (Enterprise)

```
월별 비용 내역:
  Production Tenant:     $15,000
  Dev Tenant:            $5,000
  QA Tenant:             $5,000
  Message Processing:    80만 건 (기본 100만 건 포함) → $0 추가
  B2B Messages:          5만 건 (기본 10만 건 포함) → $0 추가
  추가 B2B Partners:     20개 → $200 × 20 = $4,000
  Event Mesh Messages:   300만 건 (기본 500만 건 포함) → $0 추가
  Data Transfer:         150GB → 초과 100GB × $150 = $15,000
────────────────────────────────
  월 총계: $44,000
  연간: $528,000 (약 7.1억 원)
```

### 케이스 3: API Management만 단독 사용

```
월별 비용 내역:
  API Mgmt Pro:          $5,000
  API Calls:             800만 호출 → 초과 300만 × $1,500/100만 = $4,500
────────────────────────────────
  월 총계: $9,500
  연간: $114,000 (약 1.5억 원)
```

## 할인 구조

### CPEA 할인

```
연간 크레딧 커밋먼트:
  $200K 미만:  0~10% 할인
  $200K~500K:  10~20% 할인
  $500K~1M:    20~30% 할인
  $1M 이상:    30~50% 할인 (협상)
```

### 장기 계약 할인

| 계약 기간 | 할인율 |
|:---:|:---:|
| 월 단위 (PAYG) | 0% (기준가) |
| 1년 구독 | 10~20% |
| 3년 구독 | 20~40% |
| 5년 구독 | 30~45% |

### RISE with SAP 포함

- RISE with SAP 계약 시 Integration Suite Subscription이 **기본 포함**될 수 있음 (계약 조건에 따라)
- 포함된 범위: 통상 Standard Edition, 메시지 20만 건/월 기본
- Enterprise Edition으로의 업그레이드는 추가 비용

## 숨은 비용 / 주의사항

| 항목 | 설명 | 연간 영향 |
|------|------|:---:|
| **아웃바운드 데이터 전송** | 타사 시스템으로 데이터 전송 시 대역폭 추가 | $10K ~ $50K |
| **SAP Cloud Connector** | On-prem 연동 시 추가 인프라 필요 (소프트웨어는 무료) | 인프라 비용 |
| **B2B Partner 수** | 거래처가 늘어날수록 Partner 라이선스 추가 | Partner × $200/월 |
| **Message Processing 초과** | 예측 못한 메시지 폭증 | 가변 |
| **운영 인력 교육** | Integration Suite 교육·자격증 | $3K ~ $10K/인 |
| **마이그레이션 비용** | PI/PO → IS 전환 시 | 일회성 $50K ~ $200K |
