# 7. settlement — 정산 관리 (기능5)

> **작성일**: 2026-07-13
> **역할**: OTT 관리자 전용 정산 관리 화면. 콘텐츠별 수익 분배, 파트너 정산 데이터를 조회·실행합니다. `xs-security.json`의 `OTT_Settlement_Admin` Role Collection이 필요합니다.

---

## 📂 모듈 개요

```
cap-app/settlement/
├── mta.yaml             ← MTA 배포 정의
├── package.json         ← npm 스크립트
├── ui5-deploy.yaml      ← UI5 빌드 설정 (깨끗함 ✅)

```

## 🔑 핵심 배포 파일들

### ① `mta.yaml` — MTA 배포 정의

```yaml
_schema-version: "3.2"
ID: comsettlement
version: 1.0.0
parameters:
  deploy_mode: html5-repo
  enable-parallel-deployments: true

modules:
  - name: com-settlement-content
    type: com.sap.application.content
    path: .
    requires:
      - name: com-settlement-repo-host
        parameters:
          content-target: true
    build-parameters:
      build-result: resources
      requires:
        - name: settlement
          artifacts:
            - settlement.zip
          target-path: resources/

  - name: settlement
    type: html5
    path: .
    build-parameters:
      build-result: dist
      builder: custom
      commands:
        - npm install
        - npm run build:cf

resources:
  - name: com-settlement-repo-host
    type: org.cloudfoundry.managed-service
    parameters:
      service: html5-apps-repo
      service-plan: app-host
      service-key: com-settlement-repo-host-key   # ← board와 동일하게 하드코딩
```

### ③ `ui5-deploy.yaml` — UI5 배포 빌드 설정

```yaml
specVersion: '3.0'
metadata:
  name: settlement
type: application
resources:
  configuration:
    propertiesFileSourceEncoding: UTF-8
builder:
  resources:
    excludes:
      - "/test/**"
      - "/localService/**"
  customTasks:
    - name: ui5-task-zipper
      afterTask: generateCachebusterInfo
      configuration:
        archiveName: settlement
        additionalFiles:
          - xs-app.json
```

> ✅ `webide-extension-task-updateManifestJson` 없음. 깨끗합니다.

## 🔐 관리자 전용 접근

settlement는 일반 사용자가 접근할 수 없는 **관리자 전용** 기능입니다:

1. **UI5 레벨**: `manifest.json` 또는 Router에서 권한 확인
2. **XSUAA 레벨**: `OTT_Settlement_Admin` Role Collection 필요
3. **CAP 레벨**: `@restrict`로 서비스 접근 제한

`xs-security.json` (approuter_ott):
```json
{
  "xsappname": "com-app-xsapp",
  "role-collections": [
    {
      "name": "OTT_Settlement_Admin",
      "description": "OTT 관리자 (기능5 정산 + 기능3 구독자 추세 분석)"
    }
  ]
}
```
