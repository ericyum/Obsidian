## Prompt Hub 에 자신의 프롬프트 등록



```python
from langchain.prompts import ChatPromptTemplate


prompt = ChatPromptTemplate.from_template(
    "주어진 내용을 바탕으로 다음 문장을 요약하세요. 답변은 반드시 한글로 작성하세요\n\nCONTEXT: {context}\n\nSUMMARY:"
)
prompt
```

```
ChatPromptTemplate(input_variables=['context'], input_types={}, partial_variables={}, messages=[HumanMessagePromptTemplate(prompt=PromptTemplate(input_variables=['context'], input_types={}, partial_variables={}, template='주어진 내용을 바탕으로 다음 문장을 요약하세요. 답변은 반드시 한글로 작성하세요\n\nCONTEXT: {context}\n\nSUMMARY:'), additional_kwargs={})])
```

### 먼저 LangSmith에서 프롬프트를 한 개 등록하고 그 프롬프트를 private에서 public으로 변경해야 한다. 변경을하는 과정에서 ID가 LangSmith에 등록되기 때문이다.



```python
from langchain import hub

# 프롬프트를 허브에 업로드합니다.
hub.push("ericyum/simple-summary-korean", prompt)
```

```
'https://smith.langchain.com/prompts/simple-summary-korean/de93adb6?organizationId=2ffce1a2-5abe-42b3-883f-3fa0330b5933'
```

다음은 Hub 에 성공적으로 업로드 된 후 출력입니다.

`아이디/프롬프트명/해시`

> 출력: '[https://smith.langchain.com/hub/teddynote/simple-summary-korean/0e296563](https://smith.langchain.com/hub/teddynote/simple-summary-korean/0e296563)'



```python
from langchain import hub
```
