# 6. BTP 요금제별 Integration Suite 사용 가능 여부

> **작성일**: 2026-07-20

---

## SAP BTP 계약 모델

SAP BTP는 크게 두 가지 계약 모델로 제공되며, Integration Suite 사용 가능 여부가 달라집니다:

```
BTP 계약 모델
├── ① BTP Enterprise Agreement (CPEA)
├── ② Pay-as-you-go (PAYG)
└── ③ BTP Free Tier (제한적)
```

---

## 모델 1: BTP Enterprise Agreement (CPEA)

### 가용성

| Integration Suite 기능 | CPEA에서 사용 가능? |
|-----------------------|:---:|
| **Cloud Integration** (iFlow) | ✅ |
| **API Management** | ✅ |
| **Event Mesh** | ✅ |
| **Trading Partner Management** (B2B/EDI) | ✅ |
| **Integration Advisor** | ✅ |
| **Edge Integration Cell** | ✅ |

### 특징

- **모든 Integration Suite 기능을 CPEA 크레딧으로 사용 가능**
- 연간 최소 커밋먼트 필요 (기업 규모에 따라 협상)
- 크레딧을 Integration Suite 외 다른 BTP 서비스(SAP Build, AI Core, HANA Cloud 등)와 함께 사용 가능
- **대기업에 가장 적합**

---

## 모델 2: Pay-as-you-go (PAYG)

### 가용성

| Integration Suite 기능 | PAYG에서 사용 가능? |
|-----------------------|:---:|
| **Cloud Integration** | ✅ |
| **API Management** | ✅ |
| **Event Mesh** | ✅ (별도) |
| **Trading Partner Management** | ✅ (Enterprise) |
| **Integration Advisor** | ✅ |
| **Edge Integration Cell** | ✅ (Enterprise) |

### 특징

- **약정 없음**, 월 단위 과금
- 신용카드 등록으로 바로 시작 가능
- **스타트업·PoC·중소기업에 적합**
- PAYG로 Standard/Enterprise 모두 구독 가능
- 단, PAYG는 구독 할인이 없어 CPEA보다 비쌀 수 있음

---

## 모델 3: BTP Free Tier

### 제한 사항

BTP Free Tier는 **Integration Suite 전체 기능을 사용할 수 없습니다**. 다만:

| 항목 | Free Tier 가능 여부 |
|------|:---:|
| Integration Suite 정식 구독 | ❌ |
| API Management (제한) | ⚠️ 일부 가능 |
| BTP Cockpit 접근 | ✅ |
| Cloud Integration 평가판 | ⚠️ Trial 계정 필요 |

### Free Tier로 가능한 것

- BTP의 다른 무료 서비스 일부 체험
- Integration Suite는 별도 **Trial 계정**을 통해 평가판 사용 가능 (30~90일)
- 실제 프로덕션 사용은 CPEA 또는 PAYG 필요

---

## Integration Suite의 특수 요금제

별도로 구매 가능한 옵션:

### 1. Integration Suite Trial (평가판)

- **30일 무료** (연장 가능)
- 모든 Capability 체험 가능
- 테넌트 1개, 제한된 용량
- 프로덕션 데이터 사용 불가

### 2. Non-Production Tenant

- Dev/Test 전용 테넌트
- 프로덕션 대비 **50~70% 할인**
- 동일 기능이지만 SLA 미적용

### 3. Integration Suite Add-ons

- Edge Integration Cell Node
- 추가 Message Processing 용량
- Premium 어댑터 (일부 특수 SAP 어댑터)

---

## 어떤 모델을 선택해야 하는가?

| 기업 유형 | 권장 모델 | 이유 |
|-----------|----------|------|
| **대기업** (SAP ERP 대규모 사용) | CPEA | 통합 크레딧 관리, 볼륨 할인 |
| **중견기업** (SAP S/4HANA 도입 중) | CPEA 또는 PAYG | 유연한 계약, 필요 시 전환 |
| **스타트업** (SAP BTP 첫 입문) | PAYG | 약정 부담 없음, 월 단위 |
| **SAP 파트너사** (고객 PoC) | Trial → PAYG | 무료 평가 후 전환 |
| **개발자** (학습·테스트) | Trial + Free Tier | 무료로 충분히 테스트 가능 |

---

## BTP 서비스와의 번들링

Integration Suite는 다른 BTP 서비스와 함께 구독 시 **번들 할인**이 적용될 수 있습니다:

| 번들 조합 | 예시 |
|-----------|------|
| **통합 + 자동화** | Integration Suite + SAP Build Process Automation |
| **통합 + AI** | Integration Suite + AI Core + AI Launchpad |
| **통합 + 분석** | Integration Suite + SAP Analytics Cloud |
| **통합 + 데이터** | Integration Suite + Datasphere |
| **RISE with SAP 포함** | RISE 계약에 Integration Suite 포함 가능 |

---

## Integration Suite 서비스 플랜 (Discovery Center 기준)

| 플랜 이름 | 설명 | 포함 Capability |
|-----------|------|----------------|
| **standard** | 기본 통합 기능 | Cloud Integration, API Mgmt, Integration Advisor (기본) |
| **enterprise** | 전체 기능 | standard + B2B/EDI, Event Mesh, Integration Advisor (고급), Edge Cell |
| **api-management** | API 관리만 | API Management 단독 |
| **trial** | 평가판 (30일) | 모든 기능 체험, 제한 용량 |
