# 2. approuter — 진입점과 인증

## 책임

이 폴더는 저장소의 `approuter` 관련 소스를 파일 단위로 추적한다.

## 문서 목록

- [[approuter › .cfignore 완전 분석]] — 상위 모듈의 실행 또는 설정을 보조하는 파일이다.
- [[approuter › dev › default-env.example.json 완전 분석]] — 상위 모듈의 실행 또는 설정을 보조하는 파일이다.
- [[approuter › dev › xs-app.json 완전 분석]] — 로컬 개발에서 CAP와 정적 리소스를 연결하는 라우팅 규칙이다.
- [[approuter › package.json 완전 분석]] — 상위 모듈의 실행 또는 설정을 보조하는 파일이다.
- [[approuter › xs-app.json 완전 분석]] — 운영 Approuter의 인증 정책과 CAP destination 프록시, 정적 리소스 fallback 순서를 정의한다.

## 읽는 법

1. 진입점과 선언형 계약을 먼저 읽는다.
2. 실제 handler 또는 실행 함수를 읽는다.
3. 호출하는 라이브러리와 테스트로 내려간다.
4. 환경 변수는 루트 `.env.example`과 `mta.yaml`의 운영 값을 함께 비교한다.
